# **App Name**: TileForge

## Core Features:

- Tileset Uploader: Allow users to upload image files (e.g., PNG) to be used as a collection of tiles, displaying a preview of the tileset.
- Tile Size Configuration: Users can input and adjust the individual tile width and height (in pixels) for the currently selected tileset.
- Canvas Setup: Users can define the tilemap's dimensions by specifying its width and height in number of tiles.
- Tile Placement Tool: Provide a painting tool that enables users to select a tile from the uploaded tileset and place it onto the canvas grid at the desired location.
- Eraser Tool: A tool for users to clear or remove individual tiles from the tilemap canvas.
- Export as PNG: Allow users to export the current tilemap canvas as a single PNG image file.
- Export as JSON: Enable users to export the tilemap's underlying data structure (e.g., tile indices array) as a JSON file.

## Style Guidelines:

- Color scheme: A professional light theme to maintain focus and clarity in the editing environment. The palette aims for a balance of modern and functional.
- Primary color: A clear, confident blue (#2465AA) used for active elements, buttons, and selected states. It is professional without being dull, chosen to reflect digital tools and precision.
- Background color: A very light, cool-toned grey (#EDF3F7) derived from the primary blue, ensuring a clean and unobtrusive workspace that allows the content to stand out.
- Accent color: A vibrant, analogous cyan-blue (#44C6E2) to highlight key interactions, provide feedback, or draw attention to notifications. This color provides contrast and a modern touch.
- Headline and body font: 'Inter', a grotesque-style sans-serif font chosen for its modern, objective, and highly legible characteristics, suitable for various UI elements in a functional application.
- Use a set of clear, outline-style vector icons that are intuitive and consistent for all tools and actions within the editor.
- Employ a structured, multi-panel layout with a prominent central canvas area, accompanied by distinct sidebars for tools, settings, and asset management for efficient workflow.
- Implement subtle and quick visual feedback animations for actions like tile selection, tile placement, and tool switching to enhance user experience without causing distraction.
